Developer API

Integrate in minutes. Train at scale.

Query any environment, generate thousands of variations, and export directly to your simulator — all from a single Python call. imagine.io slots into your existing training pipeline without friction.

Python SDK with async batch generation support
Native export to Isaac Sim, MuJoCo, and OpenUSD
Parametric variation — lighting, layout, materials, physics
Environments ready in <5 minutes from query to simulator
train_pipeline.py 
import imagineo

# Initialise client
client = imagineo.Client(api_key="YOUR_API_KEY")

# Query sim-ready environments
envs = client.environments.query(
    category="kitchen",
    sim_ready=True,
    export_format="usd"
)

# Generate 10,000× variations
batch = client.generate(
    environment_id=envs[0].id,
    count=10_000,
    params={
        "lighting": "randomized",
        "layout":   "parametric",
        "materials": "varied",
    }
)

# Export to NVIDIA Isaac Sim
client.export(batch, target="isaac_sim")
# ✓ Ready in <5 minutes
